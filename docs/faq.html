Troubleshooting

Save Screenshot is not screenshotting the correct display / black screen

You may run into this issue if your computer has 2 or more GPUs. Most modern laptops comes with 2 GPUs:


  • An Integrated GPU to run your desktop / main screen
  • A discrete GPU (NVIDIA / AMD etc) to run apps

This happens when Windows chooses to run LAT on a different GPU than the one that is running your display. For example, if your main display uses Integrated Graphics and Windows chooses to run LAT using your discrete GPU (NVIDIA / AMD), then LAT would not be able to detect your main display.


This is a Windows problem that LAT cannot fix, and is faced by other apps that are widely used as well such as OBS (Open Broadcaster Software).


Use this guide by OBS to specify the GPU that LAT runs on. You should specify the same GPU that is running your main display, which is most likely your integrated graphics.


Instead of the obs64 executable as shown in the guide, choose the live-annotation-tools.exe at your LAT install location instead. When setting your Graphics Preference mode in Step 3, choose the Power Saving mode to tell Windows to use Integrated Graphics to run LAT.
